Quad RF Android App – User Manual

Quick Start

  1. Connect the Quad RF board to your phone via USB-C.
  2. Open the app. Grant USB permission when prompted.
  3. Tap "Start SDR" to begin receiving data.
  4. Use the frequency control to tune to a station.
  5. Select a demodulation mode (AM, FM, SSB).

Features

  • Waterfall Display – Shows the RF spectrum over time.
  • Spectrum Analyser – Real-time FFT of the received signal.
  • Demodulation – AM, FM, SSB (LSB/USB), and CW.
  • Recording – Save IQ data and audio recordings.

Troubleshooting

If the app does not detect your Quad RF board:

  • Check that your phone supports USB Host mode.
  • Try a different USB-C cable.
  • Ensure the board is powered.
